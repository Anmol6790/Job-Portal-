Resume Format Guidelines

Font & Layout

  • Use professional fonts: Arial, Calibri, or Times New Roman
  • Font size: 10-12pt for body text, 14-16pt for headings
  • Keep margins between 0.5-1 inch on all sides
  • Use consistent spacing throughout

Structure

  • Include clear section headings
  • Use bullet points for easy reading
  • Limit to 1-2 pages maximum
  • Organize information in reverse chronological order

Resume Keywords Guide

Essential Keywords

  • Include job-specific skills and qualifications
  • Use industry-standard terminology
  • Mention relevant certifications and licenses
  • Highlight technical skills and software proficiency

Action Verbs

  • Managed, Led, Developed, Created
  • Implemented, Designed, Analyzed, Improved
  • Coordinated, Organized, Planned, Executed
  • Resolved, Optimized, Streamlined, Enhanced

Proofreading Checklist

Content Review

  • Check for spelling and grammar errors
  • Verify all dates and numbers
  • Ensure consistent formatting
  • Review contact information accuracy

Final Steps

  • Read aloud for flow and clarity
  • Get feedback from others
  • Check for ATS compatibility
  • Verify all links work (if included)

File Guidelines

File Requirements

  • Keep file size under 5MB
  • Use PDF format for best compatibility
  • Name file professionally (e.g., "John_Smith_Resume.pdf")
  • Optimize images if included

Best Practices

  • Test file opens correctly
  • Ensure text is selectable
  • Check for proper formatting on different devices
  • Include a plain text version if requested
